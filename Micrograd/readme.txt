Micrograd is a tiny autograd engine implemented in Python. 
It supports scalar-based automatic differentiation and provides the necessary tools to create 
basic neural network components like MLP(Multi-Layer Perceptron), layers and neurons.

This is a reproduction of Andrej Karpathy's MicroGrad. 
You can find the original content and a more in-depth explanation of MicroGrad here:
https://github.com/karpathy/micrograd
